概要
----

Gemini-2.0 APIを使用してコンテンツを解析し、市場影響度やトレンドを抽出するモジュールです。
リトライロジックとエラーレート監視を実装しています。

主な機能
--------

* コンテンツの意味解析
* 市場影響度の算出
* トレンドの抽出
* エンティティの特定
* リレーションシップの分析

エラーハンドリング
----------------

* リトライロジック
* エラーレート監視
* バリデーション機能
* エラー回復メカニズム

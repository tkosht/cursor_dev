概要
----

Neo4jデータベースとの接続を管理し、トランザクション制御を提供するモジュールです。
エラーハンドリングとクリーンアップ処理を強化しています。

主な機能
--------

* データベース接続管理
* トランザクション制御
* ノード作成・更新・削除
* リレーションシップ管理
* クエリ実行

エラーハンドリング
----------------

* 接続エラーの処理
* トランザクションのロールバック
* クリーンアップ処理
* リソース管理
